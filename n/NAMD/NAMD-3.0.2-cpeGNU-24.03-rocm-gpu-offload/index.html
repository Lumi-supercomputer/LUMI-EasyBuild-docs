    Install with the EasyBuild-user module: 

    eb NAMD-3.0.2-cpeGNU-24.03-rocm-gpu-offload.eb -r
    To access module help after installation and get reminded for which stacks and partitions the module is installed, use module spider NAMD/3.0.2-cpeGNU-24.03-rocm-gpu-offload.
